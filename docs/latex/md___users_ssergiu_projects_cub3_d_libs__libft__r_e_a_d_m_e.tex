The goal of the project is to create a library of useful functions that can be used in future projects. The library contains functions for manipulating strings, memory, and lists, as well as other common programming tasks.\hypertarget{md___users_ssergiu_projects_cub3_d_libs__libft__r_e_a_d_m_e_autotoc_md3}{}\doxysection{Installation instructions}\label{md___users_ssergiu_projects_cub3_d_libs__libft__r_e_a_d_m_e_autotoc_md3}
Clone the repository\+:
\begin{DoxyItemize}
\item {\ttfamily git clone \href{https://github.com/s-sergiu/Libft.git}{\texttt{ https\+://github.\+com/s-\/sergiu/\+Libft.\+git}}}

Cd into the directory\+:
\item {\ttfamily cd Libft}

Use make to compile the static library (libft.\+a)\+:
\item {\ttfamily make}

Or cmake\+:
\item {\ttfamily cmake -\/S . -\/B build}
\item {\ttfamily cmake -\/-\/build build}
\end{DoxyItemize}\hypertarget{md___users_ssergiu_projects_cub3_d_libs__libft__r_e_a_d_m_e_autotoc_md4}{}\doxysection{Usage examples}\label{md___users_ssergiu_projects_cub3_d_libs__libft__r_e_a_d_m_e_autotoc_md4}
To be able to use it within your project you need to link it using the {\ttfamily -\/I} when you compile your object files and at the end when linking your object files to create the static archive you need to pass {\ttfamily -\/l} (name of the library) and {\ttfamily -\/L} (location of the archive file). See example below\+:

Example\+:
\begin{DoxyItemize}
\item {\ttfamily NAME = name of main project}
\item {\ttfamily INC\+\_\+\+FLAGS = path to your include directory}
\item {\ttfamily cc -\/Wall -\/Werror -\/Wextra -\/c source.\+c -\/o source.\+o -\/I\$(INC\+\_\+\+FLAGS)}
\item {\ttfamily LIBFT\+\_\+\+OBJ = path to your libft.\+a file}
\item {\ttfamily cc -\/Wall -\/Werror -\/Wextra source.\+o -\/lft -\/L\$(LIBFT\+\_\+\+OBJ) -\/o \$(NAME)}
\end{DoxyItemize}\hypertarget{md___users_ssergiu_projects_cub3_d_libs__libft__r_e_a_d_m_e_autotoc_md5}{}\doxysection{Any dependencies or prerequisites required for the project}\label{md___users_ssergiu_projects_cub3_d_libs__libft__r_e_a_d_m_e_autotoc_md5}
